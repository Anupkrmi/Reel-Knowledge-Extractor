# 🎬 Reel Knowledge

Reel Knowledge is a movie-focused web platform that blends entertainment with learning. Users can explore films, discover interesting facts, test their movie knowledge, and engage with interactive content around cinema. Because apparently humans decided watching movies wasn’t enough and needed quizzes too.

---

## 🚀 Features

- 🔍 Search movies and explore details
- 🧠 Movie trivia and knowledge quizzes
- 🎞️ Trending and popular movie recommendations
- 👤 User authentication and profiles
- ❤️ Save favorite movies
- 📱 Responsive design for mobile and desktop
- 🌙 Clean and modern UI

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript
- React.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### APIs
- TMDB API / OMDb API
